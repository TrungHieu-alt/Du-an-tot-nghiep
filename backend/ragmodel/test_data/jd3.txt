JOB TITLE
Data Engineer

LOCATION
Hanoi, Vietnam

JOB DESCRIPTION
We are looking for a Data Engineer to build and maintain our data infrastructure. You will design ETL pipelines, work with large datasets, and enable the analytics and ML teams to access clean, reliable data.

REQUIREMENTS
- 3+ years data engineering experience
- Strong Python and SQL skills
- Experience with Apache Airflow for pipeline orchestration
- Familiarity with big data tools (Spark, Kafka, or similar)
- Experience with cloud data warehouses (BigQuery, Redshift, or Snowflake)
- Knowledge of dbt for data transformation
- Docker and cloud deployment experience (AWS or GCP)

RESPONSIBILITIES
- Build and maintain ETL/ELT data pipelines
- Monitor pipeline reliability and data quality
- Migrate and optimize data warehouse schemas
- Collaborate with ML engineers on feature pipelines
- Document data lineage and transformations

TECH STACK
Python, Apache Airflow, Apache Spark, BigQuery, dbt, Kafka, Docker, AWS
